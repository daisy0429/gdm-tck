## 目标

以LDBC社交网络图模型为业务数据底座，构建一套端到端的集成测试套件。LDBC SNB 提供了真实的社交网络拓扑（Person、Post、Comment、Forum、Tag 等），天然覆盖复杂多跳关系、稠密子图、异构标签等场景，适合用来验证图库在真实业务负载下的完整功能表现。

## 覆盖范围

本目录下的测试应贯穿以下所有功能模块（对应 features/ 01~17），每项均基于 LDBC 数据集执行：

### 数据建模与写入
- [ ] DML：节点/关系的 CREATE / MERGE / DELETE / SET / REMOVE，包含批量写入与 UPSERT 语义
- [ ] 元数据：Label、RelationshipType、Property 的定义与查询

### 查询能力
- [ ] 原始 Cypher 子句：MATCH / OPTIONAL MATCH / WHERE / RETURN / ORDER BY / SKIP / LIMIT / WITH / UNWIND / UNION / CALL {} 等
- [ ] GQL：GQL 语法扩展的查询与路径表达式

### 性能加速
- [ ] 索引：单字段索引、复合索引、全文索引的创建/删除/重建；索引生效验证；索引重启后语义恢复
- [ ] 约束：UNIQUE / NODE KEY / NOT NULL / 属性存在性约束的创建、违反拦截与级联行为
- [ ] 考虑向量索引 

### 事务与并发
- [ ] 事务：显式事务 BEGIN / COMMIT / ROLLBACK；事务隔离级别；读写冲突与死锁检测
- [ ] 并发：多线程/多会话并发读写一致性；乐观锁与悲观锁行为

### 运维管理
- [ ] 导入/导出：基于 LDBC 数据的 bulk import 与 export；格式兼容性（CSV / JSON）；增量导入
- [ ] 备份/恢复：全量备份、增量备份、指定时间点恢复；备份完整性校验
- [ ] CLI：gdm-cli 命令行工具的增删改查与管理操作
- [ ] 管理后台：用户管理、数据库管理、参数配置等 admin 操作
- [ ] 任务管理：后台任务的创建、查询、取消与状态流转

### 安全与多用户/租户
- [ ] 安全：用户/角色/权限的 RBAC 模型；图级 / 标签级 / 属性级的细粒度访问控制；TLS 连接加密
- [ ] 字符集：多语言字符（中文、Unicode）在节点属性与查询中的正确存储与检索

### 高级特性
- [ ] 服务端编程：存储过程 / UDF 的注册、调用与权限控制
- [ ] 分布式：分片策略、跨分片查询、分布式事务一致性
- [ ] 可观测性：慢查询日志、执行计划分析、统计信息、运行指标采集
- [ ] 图计算：基于 LDBC 拓扑的 PageRank / 最短路径 / 连通分量等图算法验证

## 数据准备

- 使用 LDBC SNB Datagen 生成标准数据集（scale factor 可配置，建议从 SF0.1 起步）
- 提供 `.feature` 文件中的 Given 步骤完成数据导入与索引预建
- 数据集版本与生成参数记录在配置文件中，确保测试可复现
